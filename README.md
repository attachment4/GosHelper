# ГосПомощник — Проектная документация

ИИ-консультант по законодательству РФ для Android

---

## 📁 Структура папки

```
gospomoshnik/
├── screens/
│   ├── 01_main_menu.html      — Главное меню (категории, поиск, история)
│   ├── 02_chat.html           — Чат с ИИ (диалог, стриминг, карточки действий)
│   ├── 03_document.html       — Генератор документов (шаблон + PDF)
│   └── 04_subscription.html  — Экран подписки Pro (ЮKassa, СБП)
│
├── architecture/
│   └── architecture.html      — Полная архитектурная диаграмма (MVVM + Clean)
│
└── README.md                  — Этот файл
```

---

## 🏗️ Архитектура

**Паттерн:** MVVM + Clean Architecture  
**Язык:** Kotlin  
**UI:** Jetpack Compose  

```
UI (Compose)
    ↕ StateFlow / Events
ViewModel (Hilt DI + Coroutines)
    ↕ Use Cases
Domain (чистый Kotlin)
    ↕ Repository interfaces
Data (Retrofit + Room + ЮKassa SDK)
    ↕
Внешние сервисы: GigaChat API / Room DB / ЮKassa
```

## 🛠 Стек технологий

| Слой | Технология |
|------|-----------|
| UI | Jetpack Compose |
| Архитектура | MVVM + Hilt DI |
| Асинхронность | Kotlin Coroutines + Flow |
| Сеть | Retrofit + OkHttp |
| Локальная БД | Room |
| ИИ | GigaChat API (Сбер, 1М токенов free) |
| Платежи | ЮKassa Android SDK + СБП |
| PDF | iTextG |
| Бэкенд | FastAPI (Python) на Amvera |
| Дистрибуция | RuStore + APK зеркало |

## 💰 Монетизация

- **Freemium:** 10 запросов/мес бесплатно
- **Pro месяц:** 199 ₽/мес
- **Pro год:** 990 ₽/год (83 ₽/мес, экономия 58%)
- **Комиссия ЮKassa:** 3.5%
- **При 100 подписчиках:** ~17 300 ₽/мес чистыми

---

## 📋 Публикация в RuStore

1. Зарегистрироваться на console.rustore.ru
2. Оформить ИП (обязательно с февраля 2026 для монетизации)
3. Подготовить: иконку 512×512, скриншоты, описание
4. Пройти модерацию (Лаборатория Касперского + ручная)
5. Выбрать категорию: **Бизнес и финансы → Юридические сервисы**
