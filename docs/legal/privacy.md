# Политика конфиденциальности

_Редакция от 11 июня 2026 г._

## 1. Общие положения
1.1. Настоящая Политика описывает, какие данные обрабатывает приложение «ГосПомощник» (далее — «Приложение») и как обеспечивается их защита.

1.2. Приложение спроектировано по принципу **минимизации данных** и **не собирает персональные данные** пользователя в смысле Федерального закона № 152-ФЗ «О персональных данных» (ФИО, паспорт, адрес, телефон и т. п.).

## 2. Какие данные обрабатываются
2.1. **Локально на устройстве** (не покидают устройство):
- история чатов и сгенерированные документы — хранятся в локальной базе данных Приложения;
- настройки (тема, размер шрифта).

2.2. **Анонимный идентификатор устройства** — случайный обезличенный код (UUID), создаётся при первом запуске. Не связан с личностью пользователя, аппаратными идентификаторами или аккаунтами. Используется только для учёта лимита бесплатных запросов и статуса подписки.

2.3. **Текст вопросов** передаётся для генерации ответа во внешний сервис ИИ (GigaChat, ПАО «Сбербанк») по защищённому соединению. Не указывайте в вопросах персональные данные (паспорт, СНИЛС, номера карт и т. п.).

2.4. **Платёжные данные** при оформлении подписки обрабатываются платёжным сервисом **ЮKassa** на его стороне. Приложение не получает и не хранит данные банковских карт.

## 3. Что НЕ обрабатывается
- ФИО, паспортные данные, адрес, телефон, e-mail;
- геолокация;
- контакты, фото, файлы устройства;
- биометрия.

## 4. Передача третьим лицам
4.1. Данные не продаются и не передаются третьим лицам, за исключением технически необходимой передачи текста запроса сервису ИИ и платёжных данных — платёжному сервису, исключительно для оказания соответствующей услуги.

## 5. Хранение и удаление
5.1. История и документы хранятся локально и удаляются при удалении Приложения или очистке его данных.

5.2. Пользователь в любой момент может удалить историю чатов в Приложении.

## 6. Защита
6.1. Передача данных по сети осуществляется по защищённому протоколу TLS. Локальные данные защищены средствами операционной системы Android (изоляция данных приложения).

## 7. Дети
7.1. Приложение не предназначено для лиц младше 14 лет и не собирает целенаправленно данные таких лиц.

## 8. Изменения
8.1. Политика может обновляться. Актуальная редакция доступна в Приложении и в репозитории проекта на GitHub.
