# Политика графики и ресурсов

## Формат изображений

| Тип графики | Формат | Почему |
|-------------|--------|--------|
| Иконки, логотипы, простые иллюстрации | **Vector Drawable (XML)** | Масштабируется без потерь, вес ~1 КБ, один файл на все плотности |
| Фото, скриншоты, сложные растровые изображения | **WebP (lossless)** | Сжатие без потерь, на 20–35% меньше PNG, поддержка с Android API 18 (у нас minSdk 26 — везде работает) |

**Почему не PNG/JPG:** JPG теряет качество (артефакты), PNG весит больше WebP при том же качестве.

**Почему WebP, а не AVIF:** AVIF сжимает ещё лучше, но нативно декодируется только с Android 12 (API 31).
При minSdk 26 это исключило бы старые устройства, поэтому универсальный выбор — **WebP lossless**.

### Как конвертировать в WebP
Android Studio: правый клик по PNG/JPG в `res/` → **Convert to WebP** → выбрать **Lossless**.
Или CLI: `cwebp -lossless input.png -o output.webp`

## Текущее состояние
- Все иконки приложения и категорий — векторные (Material Icons + кастомный launcher).
- Растровых изображений в проекте нет → требование «без PNG/JPG» соблюдено.
- Любые будущие фото/скриншоты класть в `res/drawable` в формате **.webp (lossless)**.

## Системные промпты ИИ
Промпты консультанта вынесены в `app/src/main/assets/prompts/*.md`:
`_base.md` (общая основа) + по файлу на каждый отдел (gibdd, zhkh, labor, benefits, court, documents, general).
Редактируются без пересборки логики — `PromptRepository` собирает основу + профиль и подставляет текущую дату.
