# Отзыв согласия на обработку персональных данных

## Право
Субъект ПДн вправе **отозвать согласие** в любой момент (**152-ФЗ**). После отзыва оператор обязан прекратить обработку (с законными исключениями).

## Структура
1. Кому (оператор/организация), от кого (ФИО, контакты).
2. Заявление об отзыве согласия на обработку ПДн, ссылка на 152-ФЗ.
3. Требование прекратить обработку и уничтожить данные (если нет иных оснований хранения).
4. Дата, подпись.

## ⚠️ Важно знать
- Часть данных оператор вправе хранить по закону (бухгалтерия, договоры) — это законно.
- Направляйте заказным с описью — доказательство.
