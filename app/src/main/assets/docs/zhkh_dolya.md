# Доля в квартире: продажа и выдел

## Преимущественное право
При продаже доли другие сособственники имеют **преимущественное право покупки** (**ст. 250 ГК РФ**): сначала предложите им по той же цене (письменно, через нотариуса), срок на ответ — 1 месяц.

## Выдел доли
- В натуре (если технически возможно) или денежной компенсацией через суд.

## ⚠️ Важно знать
- Сделки с долями недвижимости — через **нотариуса**.
- Нарушение преимущественного права — другой сособственник может через суд перевести права покупателя на себя (3 месяца).
