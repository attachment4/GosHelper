# Управление без прав

## Ситуации (ст. 12.7 КоАП РФ)
- **Не получал права** — крупный штраф, авто может быть задержано.
- **Забыл права дома** — более мягкая ответственность (предупреждение/небольшой штраф), нужно подтвердить наличие ВУ.
- **Лишён прав** — крупный штраф / арест / обязательные работы.

## ⚠️ Важно знать
- Передача управления лицу без прав — отдельный штраф для владельца.
- При задержании авто — расходы на эвакуацию и стоянку; вернуть можно после устранения причины.
