# Незаконное увольнение

## Признаки незаконности
- Уволили без основания из **ст. 81 ТК РФ** или с нарушением процедуры.
- Принудили написать «по собственному» при фактическом сокращении.
- Уволили в запрещённый период (отпуск, больничный, беременность — **ст. 261 ТК РФ**).

## Что делать (срок — 1 месяц!)
1. Запросите копии приказа и трудовой/сведения о трудовой деятельности.
2. Обращение в **ГИТ (онлайнинспекция.рф)** и/или иск в суд по **ст. 392 ТК РФ** — **1 месяц** с даты увольнения.
3. Требуйте: восстановление, **оплату вынужденного прогула**, компенсацию морального вреда.

## ⚠️ Важно знать
- По искам о восстановлении **госпошлина не платится** (**ст. 333.36 НК РФ**).
- Пропуск месячного срока — частая фатальная ошибка; при уважительной причине срок восстанавливают.
