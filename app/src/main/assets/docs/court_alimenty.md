# Алименты: взыскание

## Способы
- **Соглашение** об уплате (у нотариуса) — имеет силу исполнительного листа.
- **Судебный приказ** — для долей от дохода, быстро и с пониженной пошлиной.
- **Иск** — если есть спор или нужна твёрдая сумма.

## Размер (по ГК/СК РФ)
- На 1 ребёнка — 1/4 дохода, на 2 — 1/3, на 3 и более — 1/2; либо твёрдая сумма.

## ⚠️ Важно знать
- По алиментам **госпошлина не платится** (**ст. 333.36 НК РФ**).
- За долг по алиментам — неустойка, ограничение прав, выезда, уголовная ответственность (**ст. 157 УК РФ**).
- Алименты можно взыскать за прошлый период (до 3 лет) при попытках получить их ранее.
