# Расторжение брака (развод)

## Где разводят
- **ЗАГС** — по взаимному согласию без несовершеннолетних детей (или в особых случаях).
- **Суд** — при наличии детей или несогласии одного из супругов (**ст. 21–23 СК РФ**).

## Шаги (через суд)
1. Иск о расторжении брака мировому/районному судье.
2. Госпошлина (*проверьте размер*).
3. Решение, регистрация развода в ЗАГСе.

## ⚠️ Важно знать
- Можно одновременно решить вопросы алиментов, раздела имущества, места жительства детей.
- При беременности жены/ребёнке до 1 года муж не может развестись без её согласия (**ст. 17 СК РФ**).
