# Испытательный срок

## Правила (ст. 70–71 ТК РФ)
- Устанавливается соглашением, прописывается в договоре; максимум — **3 месяца** (для руководителей до 6).
- Нельзя устанавливать беременным, несовершеннолетним, молодым специалистам по ряду оснований.

## Увольнение на испытании
- Работник может уйти, предупредив за **3 дня**.
- Работодатель — тоже за 3 дня, но **с указанием причин** (неудовлетворительный результат), письменно.

## ⚠️ Важно знать
- На испытании действуют все нормы ТК (зарплата не ниже, отпуск копится).
- «Не прошёл испытание» без обоснования можно оспорить в суде.
