# Отпуск без сохранения зарплаты (за свой счёт)

## Правила (ст. 128 ТК РФ)
- Предоставляется по заявлению работника по согласованию с работодателем.
- В ряде случаев работодатель **обязан** дать (свадьба, рождение ребёнка, смерть близкого — до 5 дней; работающим пенсионерам, инвалидам — установленное число дней в году).

## ⚠️ Важно знать
- «Принудительный» отпуск за свой счёт по инициативе работодателя — **незаконен** (это скрытый простой, подлежит оплате).
- Длительный отпуск за свой счёт сверх 14 дней в год уменьшает стаж для ежегодного отпуска.
