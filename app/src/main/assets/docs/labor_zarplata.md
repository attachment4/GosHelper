# Невыплата заработной платы

Зарплата выплачивается не реже чем каждые полмесяца (**ст. 136 ТК РФ**).

## Что делать
1. Письменное требование работодателю о выплате.
2. При задержке свыше 15 дней — можно приостановить работу (**ст. 142 ТК РФ**), уведомив письменно.
3. Жалоба в трудовую инспекцию — онлайнинспекция.рф.
4. Суд: срок по зарплатным спорам — **1 год** (**ст. 392 ТК РФ**). Положена компенсация (**ст. 236 ТК РФ**).

## Источники
- Роструд — онлайнинспекция.рф
- ТК РФ — pravo.gov.ru
