# Аварийное жильё и расселение

## Как признают аварийным
Межведомственная комиссия по заявлению жильцов/собственника обследует дом и выносит заключение; решение принимает орган местного самоуправления.

## Права при расселении
- Собственнику — другое равнозначное жильё или **выкупная цена** (рыночная + убытки).
- Нанимателю соцжилья — другое благоустроенное жильё по нормам.

## Шаги
1. Заявление в межведомственную комиссию о признании дома аварийным.
2. Экспертиза состояния дома.
3. Включение в программу расселения.

## ⚠️ Важно знать
- Выкупную цену можно оспорить, если занижена (своя оценка, суд).
- Сроки расселения — по региональной программе.
