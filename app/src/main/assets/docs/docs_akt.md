# Акт (о нарушении, заливе, недостатках)

## Зачем
Акт — доказательство факта (залив, брак, нарушение, состояние имущества).

## Что указать
- Дата, место, кто составил (ФИО, статус), присутствующие/свидетели.
- Описание факта: что обнаружено, причины, повреждения.
- Подписи всех участников (отказ от подписи — отметить).

## ⚠️ Важно знать
- Чем больше подписей и фото — тем сильнее доказательство.
- Один экземпляр — заинтересованной стороне.
