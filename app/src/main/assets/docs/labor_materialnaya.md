# Материальная ответственность работника

## Правила (гл. 39 ТК РФ)
- По общему правилу — **в пределах среднего месячного заработка**.
- Полная ответственность — только в установленных законом случаях (договор о полной матответственности, недостача вверенных ценностей, ущерб умышленно/в опьянении).

## ⚠️ Важно знать
- Работодатель обязан установить размер ущерба и причины (инвентаризация, объяснение работника).
- Нельзя удерживать больше 20% зарплаты за раз (по общему правилу).
- Упущенная выгода с работника не взыскивается — только прямой действительный ущерб.
