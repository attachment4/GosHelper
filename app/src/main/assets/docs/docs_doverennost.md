# Доверенность

## Виды
- Простая письменная (для многих бытовых действий).
- **Нотариальная** — обязательна для сделок с недвижимостью, в суде по ряду действий, передоверия.

## Что указать
- Дата и место (без даты — ничтожна), данные доверителя и представителя, перечень полномочий, срок (если не указан — 1 год), подпись.

## ⚠️ Важно знать
- Доверенность можно **отозвать** в любой момент (уведомив представителя и третьих лиц).
- Передоверие допускается, только если это прямо разрешено.
