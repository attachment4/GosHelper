# Согласие (на обработку данных / выезд ребёнка и т.п.)

## Что указать
- Кто даёт согласие (ФИО, паспорт), кому, на что конкретно, срок действия, право отзыва, дата, подпись.

## Частые виды
- Согласие на обработку персональных данных (**152-ФЗ**).
- Нотариальное согласие на выезд несовершеннолетнего за границу с одним родителем/третьим лицом.
- Согласие супруга на сделку с общим имуществом (нотариальное).

## ⚠️ Важно знать
- Согласие на обработку ПДн можно **отозвать** письменно.
- Для сделок с недвижимостью и выезда детей часто нужна нотариальная форма.
