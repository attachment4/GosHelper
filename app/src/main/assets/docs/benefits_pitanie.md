# Бесплатное питание в школе

## Кому
- **Все ученики 1–4 классов** — горячее питание бесплатно (федеральная мера).
- Дополнительно — льготным категориям (многодетные, малоимущие, инвалиды, дети участников отдельных программ) по региональным правилам, включая 5–11 классы.

## Как оформить
- Для льготных категорий — заявление в школу с подтверждающими документами.

## ⚠️ Важно знать
- Качество и организация питания контролируются; жалобы — в школу, управление образования, Роспотребнадзор.
- Региональные меры расширяют федеральные — уточняйте.
