# Перепланировка квартиры

## Что требует согласования
Изменение конфигурации (снос/перенос стен, проёмы, перенос мокрых зон) — по **гл. 4 ЖК РФ**.

## Как узаконить
1. Проект перепланировки.
2. Согласование с органом местного самоуправления (через МФЦ/Госуслуги).
3. Выполнение работ, акт приёмки, внесение изменений в техдокументацию (Росреестр).

## ⚠️ Важно знать
- Незаконная перепланировка — штраф и требование вернуть как было; проблемы при продаже.
- Нельзя затрагивать несущие конструкции без расчётов и согласования; нельзя переносить «мокрые» зоны над жилыми комнатами соседей.
