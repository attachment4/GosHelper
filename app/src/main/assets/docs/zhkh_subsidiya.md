# Субсидия на оплату ЖКУ

Если расходы на ЖКУ превышают допустимую долю в доходе семьи — государство компенсирует часть платы (**ст. 159 ЖК РФ**).

## Порог расходов
- Федерально — **22%** от совокупного дохода семьи. В регионах ниже: Москва — 10%, СПб — 14% и др. (*уточните по региону*).

## Как оформить
1. Госуслуги → поиск «субсидия на оплату ЖКУ», либо соцзащита/МФЦ.
2. Документы: о доходах семьи, о составе семьи, начисления по ЖКУ, отсутствие крупного долга (или соглашение о рассрочке).
3. Назначается обычно на **6 месяцев**, затем продление.

## ⚠️ Важно знать
- При наличии долга по ЖКУ субсидию могут приостановить — оформите рассрочку.
- Считается весь доход семьи; отдельные выплаты в доход не входят.
