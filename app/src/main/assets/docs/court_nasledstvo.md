# Наследство: как принять

## Срок
Принять наследство нужно в течение **6 месяцев** со дня смерти (**ст. 1154 ГК РФ**).

## Шаги
1. Обратитесь к нотариусу по месту открытия наследства, подайте заявление.
2. Соберите документы (свидетельство о смерти, родство, на имущество).
3. Оплатите госпошлину (зависит от родства и стоимости).
4. Получите свидетельство о праве на наследство, зарегистрируйте право.

## ⚠️ Важно знать
- Пропустили срок — восстановление через суд при уважительной причине.
- Принимая наследство, принимаете и **долги** наследодателя (в пределах стоимости имущества).
- Есть обязательная доля для нетрудоспособных иждивенцев.
