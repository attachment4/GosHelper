# Оформление пенсии

## Виды
- Страховая пенсия по старости (нужны возраст, стаж и пенсионные коэффициенты).
- По инвалидности, по потере кормильца, социальная.

## Как оформить
1. Проверьте стаж и коэффициенты в личном кабинете на Госуслугах/СФР.
2. Подайте заявление через Госуслуги или СФР (часто назначается проактивно).
3. Выберите способ доставки пенсии.

## ⚠️ Важно знать
- Можно заранее запросить выписку из лицевого счёта и **исправить «потерянный» стаж** (документами от работодателей/архивов).
- Отложенный выход повышает размер пенсии (повышающие коэффициенты).
