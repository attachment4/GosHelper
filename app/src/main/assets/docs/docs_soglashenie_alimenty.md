# Соглашение об уплате алиментов

## Суть
Нотариальное соглашение между плательщиком и получателем (**гл. 16 СК РФ**). Имеет силу **исполнительного листа** — при неуплате можно сразу нести приставам.

## Что указать
- Размер (не ниже, чем по закону при детях), порядок и сроки уплаты, индексация, ответственность.

## ⚠️ Важно знать
- Без нотариуса соглашение недействительно.
- Размер по соглашению не может быть ниже того, что причиталось бы по суду.
- Изменить/расторгнуть — также нотариально или через суд.
