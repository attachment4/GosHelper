# Замена водительского удостоверения

## Когда менять
- Истёк 10-летний срок, смена ФИО, утеря/порча, открытие новой категории.

## Как заменить
1. Госуслуги → «Замена водительского удостоверения».
2. Медсправка (форма 003-в/у) — при истечении срока обязательна.
3. Оплатите госпошлину (на Госуслугах — со скидкой), выберите подразделение/МФЦ.
4. Получите новое ВУ.

## ⚠️ Важно знать
- Срок действия можно продлить заранее — новое ВУ будет на 10 лет с даты выдачи.
- При смене фамилии старое ВУ становится недействительным.
