# Отпуск и компенсация

## Базовое
- Ежегодный оплачиваемый отпуск — **28 календарных дней** (**ст. 115 ТК РФ**); отдельным категориям больше.
- Отпускные выплачивают **не позднее чем за 3 дня** до начала отпуска (**ст. 136 ТК РФ**).

## Компенсация
- За неиспользованный отпуск — при увольнении деньгами (**ст. 127 ТК РФ**).
- Часть свыше 28 дней можно заменить деньгами по заявлению (не всем категориям).

## ⚠️ Важно знать
- «Сгорания» отпусков по закону нет — неиспользованные дни компенсируются при увольнении.
- Отзыв из отпуска — только с согласия работника.
