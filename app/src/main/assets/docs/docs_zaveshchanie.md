# Завещание

## Как составить
- Удостоверяется **нотариусом** (в исключительных случаях — иными лицами).
- Можно завещать любое имущество любым лицам, подназначить наследников, установить завещательный отказ.

## ⚠️ Важно знать
- **Обязательная доля**: несовершеннолетние/нетрудоспособные дети, нетрудоспособные супруг и родители получают не менее половины доли по закону независимо от завещания (**ст. 1149 ГК РФ**).
- Завещание можно изменить/отменить в любой момент.
- Тайна завещания охраняется законом.
