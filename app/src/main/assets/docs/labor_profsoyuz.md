# Профсоюз: права работника

## Зачем
Профсоюз защищает трудовые права, участвует в коллективных переговорах, контролирует соблюдение ТК.

## Гарантии
- Нельзя дискриминировать за членство в профсоюзе.
- Увольнение членов профсоюза по ряду оснований — с учётом мнения профоргана (**ст. 82, 373 ТК РФ**).
- Право создавать профсоюз и вступать в него (**ст. 21 ТК РФ**, ФЗ № 10-ФЗ).

## ⚠️ Важно знать
- Коллективный договор может давать больше гарантий, чем ТК.
- Давление за профсоюзную деятельность — основание для жалобы в ГИТ/прокуратуру.
