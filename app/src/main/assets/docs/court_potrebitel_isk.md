# Иск по защите прав потребителя

## Преимущества
- **Госпошлина не платится** при цене иска до 1 млн ₽ (**ст. 333.36 НК РФ**).
- Иск можно подать по своему месту жительства.
- При выигрыше — **штраф 50%** от присуждённого в пользу потребителя + неустойка + моральный вред.

## Шаги
1. Досудебная претензия продавцу/исполнителю (обязательна).
2. Иск с расчётом требований (стоимость, неустойка, моральный вред).
3. Приложите чеки, претензию, доказательства.

## ⚠️ Важно знать
- Неустойка обычно 1% в день за просрочку требований (по товарам) — считайте.
- Привлеките Роспотребнадзор для заключения в вашу пользу.
