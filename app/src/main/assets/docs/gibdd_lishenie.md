# Лишение прав и возврат водительского удостоверения

## Порядок
- Лишение назначает только **суд** (**гл. 12 КоАП РФ**). Срок течёт с момента сдачи прав в ГИБДД.
- В течение **3 рабочих дней** после вступления решения в силу нужно сдать ВУ в ГИБДД (**ст. 32.7 КоАП РФ**). Не сдали — срок не идёт.

## Как вернуть после окончания срока
1. Сдать **теоретический экзамен** в ГИБДД (для большинства оснований).
2. Оплатить все штрафы.
3. При лишении за опьянение — пройти **медкомиссию** заново.
4. По истечении не менее половины срока можно вернуть досрочно по ряду оснований (кроме «пьяных» статей).

## ⚠️ Важно знать
- Управление в период лишения — **ст. 12.7 КоАП РФ**: крупный штраф / арест / обязательные работы.
- Повторное «опьянение» — уже **уголовная ответственность (ст. 264.1 УК РФ)**.
