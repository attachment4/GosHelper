# Госномера: получение, сохранение, дубликат

## Ситуации
- Новые номера — при регистрации (можно оставить прежние при смене авто — сохранение номера).
- Дубликат при утере/краже — через изготовителя номерных знаков; при краже — заявление в полицию.

## Как сохранить свой номер
1. При продаже авто подайте заявление о сохранении номера за собой (хранится ограниченный срок).
2. При покупке нового авто — присвоение сохранённого номера.

## ⚠️ Важно знать
- «Красивые» номера через посредников за плату — риски; официально распределяются по правилам.
- Подложные/нечитаемые номера — **ст. 12.2 КоАП РФ** (вплоть до лишения).
