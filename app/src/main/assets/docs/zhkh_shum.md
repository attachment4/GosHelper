# Шумные соседи: закон о тишине

## Что регулирует
Время тишины устанавливают **региональные законы** (обычно 22:00–07:00, плюс «тихий час» днём). Ремонтные работы ограничены по времени.

## Что делать
1. Зафиксируйте нарушение (аудио/видео, свидетели).
2. Вызовите участкового/полицию (102) для фиксации.
3. Жалоба участковому или в Роспотребнадзор (при превышении норм шума).

## ⚠️ Важно знать
- Штрафы за нарушение тишины — по региональному КоАП.
- Систематические нарушения можно решать через суд (вплоть до выселения злостных нарушителей-нанимателей).
