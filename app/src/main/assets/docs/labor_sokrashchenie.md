# Сокращение: права и выплаты

## Что обязан работодатель
1. Уведомить **под подпись за 2 месяца** (**ст. 180 ТК РФ**).
2. Предложить имеющиеся подходящие вакансии.
3. Учесть **преимущественное право** оставления (**ст. 179 ТК РФ**).

## Выплаты (самый выгодный сценарий для работника)
- **Выходное пособие** — средний месячный заработок.
- Сохранение среднего заработка на период трудоустройства — **до 2 месяцев** (в исключении до 3), при учёте в ЦЗН.
- Компенсация за неиспользованный отпуск + расчёт.
- Можно уволиться досрочно с письменного согласия — с доп. компенсацией за оставшийся срок.

## ⚠️ Важно знать
- «По собственному» вместо сокращения = потеря всех этих выплат. Не подписывайте под давлением.
- Нарушение процедуры — основание оспорить и взыскать.
