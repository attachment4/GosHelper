# Договор купли-продажи

## Обязательные условия
- Стороны, **предмет** (что продаётся, идентификация), **цена**, порядок передачи и оплаты.
- Для недвижимости/авто — дополнительные данные (кадастр/VIN), для недвижимости — регистрация перехода права.

## ⚠️ Важно знать
- Недвижимость: переход права регистрируется в Росреестре; расчёты безопаснее через аккредитив/эскроу/ячейку.
- Авто: 3 экземпляра, передача ПТС/СТС; покупатель регистрирует в 10 дней.
- Проверяйте обременения (залог, арест), полномочия продавца.
