# Претензия по защите прав потребителей

Основание — Закон РФ «О защите прав потребителей» № 2300-1.

## Структура
1. Кому (продавец/исполнитель) и от кого.
2. Описание: что куплено, когда, какой недостаток.
3. Требование: возврат, замена, ремонт, возмещение.
4. Срок ответа — обычно 10 дней.
5. Дата, подпись, приложения (чек, фото).
