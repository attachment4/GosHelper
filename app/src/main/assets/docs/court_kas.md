# Оспаривание действий госоргана (КАС)

## Когда
Незаконные решения/действия/бездействие органов власти и должностных лиц оспариваются по **КАС РФ** (административное судопроизводство).

## Срок и порядок
- Обычно **3 месяца** с момента, когда узнали о нарушении права.
- Иногда обязателен досудебный (жалоба в вышестоящий орган).

## ⚠️ Важно знать
- По ряду споров с госорганами бремя доказывания законности — на органе.
- Можно просить приостановить оспариваемое решение (обеспечительные меры).
