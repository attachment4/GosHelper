# Исковое заявление: образец структуры

## Обязательные реквизиты (ст. 131 ГПК РФ)
1. Наименование суда.
2. Истец и ответчик (ФИО/наименование, адреса, контакты).
3. В чём нарушение прав, обстоятельства и доказательства.
4. Требования («Прошу…») и их обоснование (статьи).
5. Цена иска и расчёт (по имущественным).
6. Перечень приложений, дата, подпись.

## Приложения (ст. 132 ГПК РФ)
- Копии иска по числу лиц, документ об оплате госпошлины (или льгота), доказательства, досудебная претензия (если обязательна).

## ⚠️ Важно знать
- Ошибки в реквизитах/подсудности → оставление без движения/возврат.
- Подать можно лично, почтой или электронно через ГАС «Правосудие».
