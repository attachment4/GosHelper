# Судебный приказ

Упрощённый порядок для **бесспорных** требований (долг по расписке, ЖКУ, алименты в твёрдой части и т.п.) — **гл. 11 ГПК РФ**.

## Плюсы и минусы
- ➕ Госпошлина **в 2 раза меньше**, чем по иску; рассмотрение без заседания; быстро (5 дней).
- ➖ Должник может **отменить приказ** простым возражением в 10 дней — тогда нужно подавать иск.

## Шаги
1. Подайте заявление о вынесении приказа мировому судье по подсудности.
2. Приложите документы, подтверждающие требование.
3. Получите приказ; при отмене — обращайтесь в исковом порядке.

## ⚠️ Важно знать
- Лимит суммы для приказа ограничен законом (*уточните актуальный порог*).
- Приказ — сразу исполнительный документ, можно нести приставам.
