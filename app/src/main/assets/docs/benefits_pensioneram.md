# Льготы пенсионерам

## Основные меры
- Налоговые льготы (освобождение от налога на одно жильё каждого вида, льгота по земле).
- Компенсации/льготы по ЖКУ (региональные), бесплатный/льготный проезд.
- Доплаты до прожиточного минимума пенсионера, региональные надбавки.
- Бесплатная диспансеризация, льготные лекарства (для отдельных категорий).

## Как оформить
- Через Госуслуги/СФР/соцзащиту; часть назначается автоматически.

## ⚠️ Важно знать
- Работающим пенсионерам индексация была ограничена — уточняйте актуальные правила.
- Региональные льготы шире федеральных — проверяйте по своему региону.
