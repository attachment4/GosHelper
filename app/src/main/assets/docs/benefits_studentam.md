# Льготы и выплаты студентам

## Что доступно
- Стипендии (академическая, социальная, повышенная), матпомощь.
- Льготный проезд, скидки, бесплатное общежитие для нуждающихся.
- Налоговый вычет за обучение (себе или родителям — за детей).

## ⚠️ Важно знать
- Социальная стипендия — для отдельных категорий (малоимущие, сироты, инвалиды) по справке.
- Вычет за обучение — за последние 3 года, при официальном доходе с НДФЛ.
