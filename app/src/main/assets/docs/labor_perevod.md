# Перевод и изменение условий труда

## Правила
- Перевод на другую работу — **только с письменного согласия** (**ст. 72 ТК РФ**), кроме временных исключений (ЧС, простой — до 1 месяца).
- Изменение условий по инициативе работодателя (**ст. 74 ТК РФ**) — уведомление за **2 месяца** с обоснованием организационных/технологических причин.

## ⚠️ Важно знать
- Снижение зарплаты «просто так» незаконно.
- При отказе от изменений работодатель обязан предложить другие вакансии; увольнение — по специальной процедуре с выплатами.
