# Тонировка: нормы и штраф

## Норма
Светопропускание лобового и передних боковых стёкол — **не менее 70%** (техрегламент). Нарушение — **ст. 12.5 КоАП РФ**.

## Что грозит
- Штраф; инспектор может потребовать устранить. Повторные требования и неустранение — риск эскалации.

## ⚠️ Важно знать
- Замер — сертифицированным прибором при определённых условиях (температура, чистота стекла); нарушение методики — основание оспорить.
- «Снять тонировку на месте» требовать не вправе — выписывается штраф.
