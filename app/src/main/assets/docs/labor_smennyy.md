# Сменный график и ночные смены

## Правила
- Сменная работа — по графику сменности, с которым знакомят заранее (обычно за месяц).
- **Ночное время** (22:00–6:00) — доплата не менее 20% (**ст. 154 ТК РФ**).
- Запрещены 2 смены подряд; учитывается суммированный учёт рабочего времени.

## ⚠️ Важно знать
- Переработка при суммированном учёте оплачивается как сверхурочная по итогам периода.
- К ночным/сменным работам ограниченно привлекают беременных, родителей малолетних (с их согласия).
