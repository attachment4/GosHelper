# Выселение и снятие с регистрации

## Основания
- Прекращение права пользования (бывший член семьи собственника, истечение найма, долг).
- Только **через суд** (за исключением добровольного снятия с учёта).

## Шаги (для собственника)
1. Уведомление о добровольном снятии с регистрации/освобождении.
2. Иск о признании утратившим право пользования и выселении.
3. Снятие с регистрации на основании решения суда.

## ⚠️ Важно знать
- Нельзя выселить «на улицу» несовершеннолетних без учёта их прав.
- Прописка (регистрация) не даёт права собственности, но снятие — только в законном порядке.
