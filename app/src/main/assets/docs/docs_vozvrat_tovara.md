# Заявление на возврат товара

## Основание
Закон «О защите прав потребителей» № 2300-1: возврат/обмен надлежащего товара в **14 дней** (кроме перечня невозвратных), некачественного — в течение гарантийного срока.

## Структура
1. Кому (продавец), от кого.
2. Что куплено, когда, чек/доказательство покупки.
3. Причина возврата, требование (возврат денег/обмен/ремонт).
4. Реквизиты для возврата, дата, подпись.

## ⚠️ Важно знать
- Деньги возвращают обычно в течение 10 дней.
- Технически сложный товар возвращают по особым правилам; отказ можно оспорить (Роспотребнадзор, суд) + штраф 50%.
