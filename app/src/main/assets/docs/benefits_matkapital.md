# Материнский (семейный) капитal

Регулируется **ФЗ № 256-ФЗ**. Сертификат оформляется **автоматически** (проактивно) после рождения ребёнка — приходит в личный кабинет на Госуслугах.

## На что можно потратить
- Улучшение жилищных условий (покупка/строительство, погашение ипотеки).
- Образование детей.
- Ежемесячная выплата на ребёнка (при нуждаемости).
- Накопительная пенсия матери.
- Адаптация детей-инвалидов.

## Как распорядиться
1. Госуслуги → «Распоряжение материнским капиталом» → выбрать цель.
2. Приложить документы по выбранному направлению.
3. Решение СФР — в установленный срок.

## ⚠️ Важно знать
- Суммы ежегодно индексируются (*уточните актуальный размер на sfr.gov.ru*).
- Обналичивание маткапитала — **незаконно** (уголовная ответственность).
