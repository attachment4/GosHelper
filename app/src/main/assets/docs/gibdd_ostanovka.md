# Остановка инспектором ДПС: ваши права

## Как себя вести
1. Остановитесь по требованию; инспектор обязан назвать должность, фамилию, причину.
2. По требованию инспектор предъявляет служебное удостоверение.
3. Передавайте документы; можно вести видеозапись (это не запрещено).

## Основания остановки
Нарушение, ориентировки, проверка документов на постах/рейдах и др. (административный регламент МВД).

## ⚠️ Важно знать
- Выходить из авто обязательно не всегда — уточняйте причину.
- Все действия и нарушения фиксируйте на видео — пригодится при обжаловании.
- Подписывая протокол, пишите «не согласен» и свои возражения.
