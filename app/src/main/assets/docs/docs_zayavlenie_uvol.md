# Заявление на увольнение по собственному желанию

## Как написать
1. Кому (руководителю, организация), от кого (должность, ФИО).
2. Текст: «Прошу уволить меня по собственному желанию (**ст. 80 ТК РФ**) с такого-то числа».
3. Дата, подпись.

## Сроки
- Предупредить за **2 недели** (отработка); по соглашению — раньше.
- В ряде случаев увольняют без отработки (зачисление на учёбу, выход на пенсию, нарушение работодателем).

## ⚠️ Важно знать
- Подавайте в 2 экземплярах, на своём — отметка о приёме с датой.
- До истечения срока заявление можно **отозвать** (если на ваше место не приглашён другой письменно).
