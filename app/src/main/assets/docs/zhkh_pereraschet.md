# Перерасчёт за некачественные коммунальные услуги

Если услуга не оказывалась или была плохого качества (нет горячей воды, холодные батареи), вы вправе требовать **перерасчёт** (Постановление Правительства РФ № 354).

## Шаги
1. Зафиксируйте нарушение: вызовите представителя УК, составьте **акт**.
2. Подайте заявление на перерасчёт в УК (с копией акта).
3. При отказе — жалоба в Госжилинспекцию через ГИС ЖКХ.

## Источники
- ГИС ЖКХ — dom.gosuslugi.ru
- Жилищный кодекс РФ — pravo.gov.ru
