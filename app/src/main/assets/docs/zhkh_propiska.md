# Регистрация по месту жительства (прописка)

## Виды
- Постоянная (по месту жительства) и временная (по месту пребывания).

## Как оформить
1. Госуслуги → «Регистрация по месту жительства/пребывания».
2. Согласие собственника (если не свой), документы на жильё.
3. Получение отметки/свидетельства.

## ⚠️ Важно знать
- Проживание без регистрации свыше срока — штраф (**ст. 19.15.1 КоАП РФ**), кроме исключений (регистрация в том же регионе/у близких родственников).
- Регистрация не даёт права собственности; собственник может снять с учёта через суд.
