# Как подать исковое заявление

## Шаги
1. Определите ответчика и подсудность (мировой/районный суд) — **ст. 28–32 ГПК РФ**.
2. Составьте иск по **ст. 131 ГПК РФ**, приложите документы по **ст. 132 ГПК РФ**.
3. Рассчитайте и оплатите госпошлину (**ст. 333.19 НК РФ** — проверьте актуальный размер).
4. Подайте: лично, почтой или электронно через ГАС «Правосудие».

## Источники
- ГАС «Правосудие» — sudrf.ru, ej.sudrf.ru
- Кодексы — pravo.gov.ru
