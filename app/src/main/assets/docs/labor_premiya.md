# Премии и лишение премии

## Как устроено
Премия — часть системы оплаты по локальным актам (положение о премировании). Условия и показатели — в документах работодателя.

## Лишение/снижение
- Допустимо только по основаниям из положения о премировании (невыполнение показателей).
- «Депремирование» как наказание за дисциплину — спорно: дисциплинарные взыскания исчерпывающи (**ст. 192 ТК РФ**), отдельного «штрафа» нет.

## ⚠️ Важно знать
- Если премия гарантированная (входит в обязательную часть) — её снижение незаконно.
- Требуйте ознакомить с положением о премировании; необоснованное лишение оспаривается.
