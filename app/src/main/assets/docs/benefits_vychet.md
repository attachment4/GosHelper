# Налоговые вычеты (вернуть НДФЛ)

## Какие бывают
- **Имущественный** — при покупке жилья (до 260 000 ₽ с покупки + до 390 000 ₽ с процентов по ипотеке).
- **Социальный** — лечение, обучение, спорт, ДМС (лимиты по НК РФ).
- **Стандартный** — на детей.
- **Инвестиционный** — по ИИС.

## Как получить
1. Через работодателя (не удерживают НДФЛ) или по декларации **3-НДФЛ**.
2. Проще всего — **упрощённый порядок** в личном кабинете ФНС (предзаполненное заявление).

## ⚠️ Важно знать
- Вернуть можно за **последние 3 года**.
- Нужен официальный доход с НДФЛ 13%.
