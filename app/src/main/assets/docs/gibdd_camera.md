# Обжалование штрафа с камеры

## Когда есть шанс отменить
- За рулём был не собственник (продал/передал по доверенности/каршеринг).
- Ошибка распознавания номера, дубль штрафа, неверное место/время.
- Знак/разметка отсутствовали или нечитаемы.

## Шаги
1. Получите постановление (Госуслуги) и фотоматериалы.
2. Соберите доказательства (договор купли-продажи, полис ОСАГО с другим водителем, путевой лист, данные каршеринга).
3. Подайте жалобу в **10 суток** (**ст. 30.3 КоАП РФ**) — в ГИБДД, вынесшее постановление, или в суд. Можно через Госуслуги.

## ⚠️ Важно знать
- При фотофиксации действует «презумпция» владельца — доказывать невиновность нужно вам.
- Срок льготной оплаты (20 дней) на время обжалования может истечь — учитывайте риск.
