# Простой: оплата и права

## Что это
Временная приостановка работы по причинам экономического/технического характера (**ст. 72.2, 157 ТК РФ**).

## Оплата
- По вине работодателя — **не менее 2/3 среднего заработка**.
- По причинам, не зависящим ни от кого — **не менее 2/3 оклада (тарифа)**.
- По вине работника — не оплачивается.

## ⚠️ Важно знать
- «Отправили в неоплачиваемый отпуск» вместо простоя — незаконно; это скрытый простой, подлежит оплате.
- Простой оформляется приказом с указанием причин и сроков.
