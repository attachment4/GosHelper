# Льготный проезд

## Кому
Пенсионеры, инвалиды, ветераны, школьники/студенты, многодетные — по региональным правилам (соцкарта, бесплатный/льготный проезд).

## Как оформить
- Социальная карта/проездной через соцзащиту/МФЦ/Госуслуги (в зависимости от региона).
- Для НСУ-получателей — компенсация проезда к месту лечения.

## ⚠️ Важно знать
- Набор льгот сильно отличается по регионам — проверяйте локально.
- Часть категорий вправе выбрать деньги вместо натуральной льготы.
