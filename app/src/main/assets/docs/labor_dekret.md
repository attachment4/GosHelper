# Декрет и детские пособия

## Выплаты
- Пособие по беременности и родам (декретные) — по больничному, через СФР.
- Единовременное пособие при рождении ребёнка.
- Ежемесячное пособие по уходу за ребёнком до 1,5 лет.
- Единое пособие (при нуждаемости) — на детей до 17 лет.

## Как оформить
1. Декретные — через работодателя/СФР по электронному больничному.
2. Остальные — через Госуслуги/СФР; часть назначается проактивно.

## ⚠️ Важно знать
- За время отпуска по уходу сохраняется рабочее место.
- Суммы индексируются ежегодно — проверяйте актуальные на sfr.gov.ru.
