# Удалённая (дистанционная) работа

## Оформление (гл. 49.1 ТК РФ)
- Удалёнка оформляется трудовым договором/допсоглашением: постоянная, временная или периодическая.
- Сохраняются все трудовые права (зарплата, отпуск, больничный).

## Важные правила
- Работодатель обеспечивает оборудование или компенсирует использование личного.
- Нельзя снижать зарплату только из-за удалёнки.
- Перевести на удалёнку без согласия можно лишь в исключительных случаях (ЧС) — временно.

## ⚠️ Важно знать
- Право «быть офлайн» вне рабочего времени — переписка ночью не обязывает отвечать.
- Увольнение дистанционщика — по общим основаниям + специальные (например, невыход на связь без причины).
